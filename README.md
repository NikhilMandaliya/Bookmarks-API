# NestJS APIs

Simple REST API built using NestJS with TypeScript. The API is designed to handle user authentication, and bookmarks management.

## Description

Learning NestJS, TypeScript,....

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
